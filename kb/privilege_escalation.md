# Privilege Escalation Playbook

Privilege Escalation occurs when a malicious user or program gains elevated access to resources that are normally protected from an application or user.

## Recommended Remediation
1. **Isolate Asset**: Immediately disconnect the affected server or workstation from the network.
2. **Revoke Access**: Revoke the compromised account's privileges and disable the account.
3. **Audit Logs**: Review system audit logs to determine the extent of the unauthorized access.
4. **Patch Vulnerability**: Identify and patch the vulnerability (e.g., missing OS patch or misconfiguration) that allowed the escalation.
